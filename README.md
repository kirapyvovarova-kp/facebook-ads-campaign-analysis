# Facebook Ads Campaign Analysis

## Project Overview

This project analyzes Facebook advertising campaign performance using marketing data. The goal was to evaluate campaign effectiveness, identify key performance drivers, and explore relationships between marketing metrics.

## Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Tasks

* Data cleaning and validation
* Calculation of marketing metrics
* ROMI analysis
* Campaign performance evaluation
* Trend analysis using rolling averages
* Correlation analysis
* Data visualization

## Results

The analysis identified factors influencing campaign performance and return on marketing investment (ROMI). Visualizations were created to support data-driven marketing decisions and highlight trends across advertising campaigns.
